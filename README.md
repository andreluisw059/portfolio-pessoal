# Portfólio Pessoal

## Descrição

Este projeto consiste em um portfólio pessoal online, desenvolvido para apresentar minhas habilidades, projetos e informações de contato a recrutadores e empresas interessadas no meu trabalho.

## Funcionalidades

| Seção | Descrição |
|---|---|
| Header | Foto pessoal, nome, bio e links para redes sociais (GitHub, LinkedIn, etc.). |
| Projetos | Demonstração de projetos realizados, com imagens, descrições breves e links. |
| Habilidades | Lista de habilidades técnicas e interpessoais. |
| Contato | Formulário para envio de mensagens e informações de contato adicionais. |

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* Figma (para design)

## Backlog

| Épico | Feature | User Story | Critérios de Aceitação |
|---|---|---|---|
| 1: Posição dos elementos | Organização dos elementos na página. | Ao acessar o site, os elementos devem estar organizados de forma clara e intuitiva. | * Elementos dispostos em 3 colunas de grid. <br> * Responsivo em diferentes dispositivos. <br> * Código de fácil manutenção. |
| 2: Header e links sociais | Botões de links para redes sociais, foto e informações pessoais. | Ao acessar o site, os links devem ser facilmente acessíveis e as informações pessoais devem estar formatadas. | * Links fáceis de acessar. <br> * Elementos alinhados corretamente. <br> * Área com foto e informações pessoais formatadas. <br> * Compatibilidade em todos os dispositivos. |
| 3: Seção de projetos e habilidades | Demonstração de projetos e habilidades. | Ao acessar o site, os projetos e habilidades devem ser apresentados de forma clara e concisa. | * Imagem e descrição breve dos projetos com links. <br> * Habilidades em formato de lista. <br> * Responsivo em diferentes dispositivos. <br> * Código de fácil manutenção. |
| 4: Formulário de contato | Formulário para envio de mensagens. | Ao acessar o site, o formulário deve permitir o envio de mensagens. | * Campos obrigatórios. <br> * Responsivo em diferentes dispositivos. |
